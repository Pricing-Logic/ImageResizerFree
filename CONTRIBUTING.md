# Contributing to ImageResizerFree

Thanks for your interest in contributing!

## How to Suggest Features or Report Bugs

1. Go to the [Issues tab](https://github.com/Pricing-Logic/ImageResizerFree/issues)
2. Click "New Issue"
3. Describe your idea or the bug you found
4. Submit - we'll review it

## How to Contribute Code

1. **Fork** this repository
2. Create a new branch for your feature (`git checkout -b feature/my-idea`)
3. Make your changes
4. Test locally by opening `index.html` in a browser
5. Commit your changes (`git commit -m "Add my feature"`)
6. Push to your fork (`git push origin feature/my-idea`)
7. Open a **Pull Request** against this repo's `main` branch

## Guidelines

- Keep it simple - this is a lightweight, no-backend tool
- No external dependencies (no npm, no build tools)
- All processing must stay client-side (privacy first)
- Test in multiple browsers if possible

## What We're Looking For

- Bug fixes
- Accessibility improvements
- Performance optimizations
- New output formats (if browser-supported)
- UI/UX improvements
- Translations

## What We're NOT Looking For

- Server-side features
- User accounts / authentication
- Analytics or tracking
- Features that require external APIs

## Questions?

Open an issue and ask!
