iresized.com Test Suite

About These Tests

This test suite validates the functionality and security of iresized.com tools. Tests are browser-based and run entirely client-side.

  • AUTO - Automated tests that run immediately
  • MANUAL - Requires uploading test files

Available Test Suites

Security Functions AUTO

Tests for core security utilities: sanitizeFilename(), validatePositiveInt(), and XSS prevention.

Coverage: Path traversal prevention, dangerous character filtering, input validation, length limits

Run Tests

Strip EXIF/Metadata AUTO MANUAL

Tests for metadata stripping via canvas re-encoding. Includes format conversion and quality settings.

Coverage: EXIF removal, dimension preservation, format conversion, quality settings, memory cleanup

Run Tests

HEIC Converter AUTO MANUAL

Tests for HEIC/HEIF to JPEG/PNG/WebP conversion using heic2any library.

Coverage: Library loading, file validation, size limits, filename generation, ZIP creation

Run Tests

Bulk Rename AUTO MANUAL

Tests for pattern-based batch file renaming with ZIP download.

Coverage: Pattern parsing ({n}, {name}, {date}), unique name generation, file limits, ZIP creation

Run Tests

Running Tests Locally

Setup Instructions

  1. Start a local server in the project root:
    npx serve . or python -m http.server 8000
  2. Navigate to http://localhost:PORT/tests/
  3. Click "Run All Automated Tests" or open individual test pages

Manual Testing Requirements

  • Strip EXIF: A JPEG photo from a camera/phone with GPS data
  • HEIC Converter: HEIC/HEIF files from an iPhone
  • Bulk Rename: Multiple image files (any format)
